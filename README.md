### 🎯 เคล็ดลับ:

- ใช้ `##`, `###` เพื่อจัดหัวข้อ
- ใช้ ` ```bash ` เพื่อจัดโค้ดให้เป็นระเบียบ
- ใส่ลิงก์ หรือไอคอน emoji บ้าง (เช่น 🚀, ✅, 📦) เพื่อให้น่าสนใจ
- หากใช้ `nodemon` เป็นประจำ อาจแนะนำให้ใส่ใน `scripts` ของ `package.json` ด้วย เช่น:

```json
"scripts": {
  "start": "nodemon app.js"
}
